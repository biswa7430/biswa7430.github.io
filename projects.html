Geometry-Preserving Contrastive Learning for Aerial Object Detection

Status: Submitted to ICPR 2026
Duration: January 2026
Focus: Cross-Geographic Domain Generalization
Role: Lead Researcher
Key Achievement: 36% improvement on VisDrone→IndraEye transfer (34.2%→46.5% mAP)

Overview: Developed novel GPCL framework with dual-level style augmentation for aerial object detection. Implemented IoU-weighted contrastive learning and detection-specific consistency regularization combining geometric (IoU) and semantic (KL-divergence) losses. First work to address cross-geographic aerial detection with geometry-preserving domain generalization.
