## Tips for Image Optimization
1. Use JPEG for photos (80-85% quality)
2. Use PNG for logos and graphics with transparency
3. Compress images using tools like:
   - TinyPNG (https://tinypng.com/)
   - ImageOptim (Mac)
   - Squoosh (https://squoosh.app/)
4. Consider WebP format for modern browsers
5. Use responsive images with srcset for different screen sizes
